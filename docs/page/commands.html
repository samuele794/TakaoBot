Qua troverai la lista di tutti i comandi del bot.
Il prefisso usabile per i comandi varia in base al server in cui viene configurato.
Qua verranno descritti i comandi usando il simbolo predefinito.

I comandi possono essere usati taggando il bot @TakaoBot <comando>
o usando il simbolo di comando seguito dal nome.

Configurazione e Info


Info

%info
Questo comando permette il richiamo delle info del bot


Help

%help
Questo comando permette il richiamo dell'aiuto per i comandi direttamente dal bot


Configuration Command

%configurationCommand <simbolo del comando>
Questo comando permette la configurazione del prefisso per i comandi.
Questo comando è riservato al solo utilizzo da parte del proprietario del server


Black Desert Online


BDO News Start

%BDONewsStart
Questo comando permette di iscriversi al feed delle news di BDO.
Il comando deve essere lanciato sul canale su cui si desidera ricevere le news.


BDO News Stop

%BDONewsStop
Questo comando permette di disiscriversi al feed delle news di BDO.
Il comando può essere lanciato su qualunque canale.


BDO Patch Start

%BDOPatchStart
Questo comando permette di iscriversi al feed delle patch di BDO.
Il comando deve essere lanciato sul canale su cui si desidera ricevere le patch.


BDO Patch Stop

%BDOPatchStop
Questo comando permette di disiscriversi al feed delle patch di BDO.
Il comando può essere lanciato su qualunque canale.


BDO Boss Start

%BDOBossStart
Questo comando permette di iscriversi agli allarmi dei boss di BDO.
Il comando deve essere lanciato sul canale su cui si desidera ricevere i boss.


BDO Boss Stop

%BDOBossStopt
Questo comando permette disiscriversi agli allarmi dei boss di BDO.
Il comando può essere lanciato su qualunque canale


BDO Boss Table

%BDOBossTable
Questo comando ti fa vedere la tabella dei boss di BDO.


Auto Trasporti Milano ATM

ATTENZIONE: gli alert in tempo reale potrebbero non essere precisi al 100%,
in caso l'ATM non utilizzi i tag filtrati passerranno messaggio solamenti informativi.

ATM Alert Start

%ATMAlertStart
Questo comando permette di iscriversi agli avvisi dell'ATM Milano.
Il comando deve essere lanciato sul canale su cui si desidera ricevere gli avvisi.


ATM Alert Start

%ATMAlertStop
Questo comando permette disiscriversi agli avvisi dell'ATM Milano.
Il comando può essere lanciato su qualunque canale


ATM Metro Status

%ATMMetroStatus / %ATMStatoMetro
Questo comando permette di ottenere lo stato attuale della rircolazione nella metro ATM
Il comando può essere lanciato su qualunque canale
